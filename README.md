# 高等教育機関における生成AI導入の実態調査

大学教育学会 課題研究「高等教育における生成AI利用のガイドラインに関する研究」（2024〜2026年度）における全国調査の結果レポートです。

## 公開ページ

https://genai-higher-ed-jp.github.io/survey-2025-sep-oct/

## 調査票（質問項目一覧）

[質問項目一覧.pdf](質問項目一覧.pdf)

## 調査概要

- **対象**: 全国1,165の大学（4年制・短期大学）および高等専門学校
- **方法**: Googleフォームによるウェブアンケート（機関長宛）
- **調査期間**: 2025年9月10日〜10月20日
- **有効回答**: 113機関（回答率9.8%）
- **倫理審査**: 桜美林大学研究倫理審査の承認を受けて実施（倫理審査番号：25020）

## 研究体制

- 研究代表者: 田中一孝（桜美林大学）
- 研究分担者: 松下佳代（京都大学）、斎藤有吾（新潟大学）、村上正行（大阪大学）、渡邊浩一（福井県立大学）、伊藤通子（東京都市大学）、羽倉尚人（東京都市大学）、平山朋子（藍野大学）、杉山芳生（藍野大学）、澁川幸加（中央大学）、石上敬子（近畿大学）、田中孝平（北海道大学）、我喜屋まり子（京都大学）
